# Proyecto De Arquitectura

## De que trata el proyecto
El proyecto trata de como se siente una persona en un salon de clases
mediante el ambiente del mismo aula.

## Preguntas Generales Acerca del Proyecto

- ¿De que trata el proyecto?
- ¿Cual es el objetivo a alcanzar?

## Preguntas Sobre la bases de datos

- ¿Que columnas quiere eliminar y es necesario porque?
- ¿Al aplicar la prediccion que quiere ver como resultado?
