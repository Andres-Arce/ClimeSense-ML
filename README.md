## Introduccion <a name="introduccion"></a>

¿Que tan cómoda se siente una persona al estar dentro de un edificio sin aire acondicionado?

ClimeSense ML es un proyecto de Data Science enfocado en desarrollar un modelo de Machine Learning capaz de predecir el nivel de confort térmico humano dentro de edificios con ventilación natural.

El proyecto busca ir más allá de una simple predicción numérica. La intención es comprender cómo variables ambientales influyen realmente en la sensación térmica de las personas y cómo estos resultados pueden apoyar decisiones dentro de la arquitectura moderna sostenible.

Actualmente enfrentamos varios desafíos:

- El aire acondicionado consume grandes cantidades de energía.
- La arquitectura moderna busca edificios más sostenibles.
- En climas húmedos (como muchas zonas tropicales), las normas tradicionales de confort térmico presentan limitaciones.

Este proyecto pretende contribuir al diseño de espacios más eficientes energéticamente utilizando análisis de datos y Machine Learning.

---

## Objetivos del Proyecto

### Objetivo General
Desarrollar un modelo de Machine Learning que permita estimar el nivel de comodidad térmica percibida por una persona dentro de edificios sin climatización mecánica.

### Objetivos Específicos
- Analizar variables ambientales relacionadas con el confort térmico.
- Realizar limpieza y análisis exploratorio de datos (EDA).
- Construir un modelo predictivo basado en Machine Learning.
- Explorar relaciones entre humedad, ventilación y percepción humana.
- Generar conocimiento útil para arquitectura sostenible.

---

## Contexto del Problema

Los modelos tradicionales de confort térmico fueron diseñados principalmente para edificios con aire acondicionado. Sin embargo, muchos edificios actuales buscan depender de ventilación natural para reducir el consumo energético.

Esto genera una necesidad:

Comprender el confort humano en condiciones reales y variables.

El uso de Data Science permite analizar patrones complejos que modelos tradicionales no capturan fácilmente.

## Metodología <a name="metodologia"></a>

El desarrollo del proyecto sigue un flujo típico de ciencia de datos:

1. Recolección y preparación de datos
2. Limpieza de datos
3. Análisis Exploratorio de Datos (EDA)
4. Ingeniería de características
5. Entrenamiento del modelo de Machine Learning
6. Evaluación del modelo
7. Interpretación de resultados

Fases principales del proyecto:

- Limpieza de Datos: Preparación y depuración del dataset.
- EDA: Comprensión de patrones, correlaciones y comportamiento de variables.
- Modelo: Desarrollo del modelo de Machine Learning (MML).

## Dataset y Fuente de Datos

Información aún en desarrollo.

Se asume que el proyecto utilizará datos ambientales relacionados con confort térmico humano, incluyendo variables como:
